# Discord
Offentlige mesh-netværk styrkes, når vi kan koordinere og samarbejde med hinanden.

Derfor kan du finde [Danske Meshtastic Brugere](https://discord.gg/EXWWwDmfBN) på Discord, hvor alle er velkommen til at være med - *klik på linket og du bliver automatisk tilføjet*.

> [Discord](discord.com) er et chat-service, hvor du opretter en konto, som kan være tilknyttet flere servere. På hver server findes der forskellige kanaler (channels), som typisk er opdelt efter emner. Du kan både tilgå din konto via din webbrowser eller deres app.

Der er flere aktive brugere på serveren, og det er et oplagt sted at stille spørgsmål, få hjælp eller bare snakke om Meshtastic m.m.

På vores Discord kan du finde en række kanaler, herunder, men ikke begrænset til:
- *[#hardware](https://discord.com/channels/1271190392574054460/1271191110621859880)* udstyr, antenner, LoRaWAN-enheder m.m.
- *[#software](https://discord.com/channels/1271190392574054460/1271191139604627517)* snak om Meshtastic, MeshCore eller generel LoRaWAN-programmering
- *[#sjælland](https://discord.com/channels/1271190392574054460/1380282561674084568)* koordinere med folk på sjælland
- *[#midtjylland](https://discord.com/channels/1271190392574054460/1380282794357559358)* koordinere med folk i midtjylland
- *[firmware](https://discord.com/channels/1271190392574054460/1271461794837168180)* automatisk påmindelse når ny version af Meshtastic er ude samt snak af dem
